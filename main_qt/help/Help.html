VAL 508 permite controlar los tiempos de activación de hasta 8 valvulas.

El menú de Archivo permite guardar, abrir, crear un archivo y limpiar todos los campos de programación de las válvulas.

El menú de Programa ofrece control sobre la conexión con el hardware que controla las válvulas. Las operaciones posibles son: ejecutar, detener y cerrar conexión.

El menú de Ayuda contiene información acerca del programa y el manual de usuario.
