Archivo|Nuevo Crea un nuevo archivo e inicializa las casillas de programación a cero.

Archivo|Abrir Abre un archivo existente compatible.

Archivo|Guardar Guarda los datos actuales al archivo actual, en caso de no haber un archivo actual se comporta como Guardar Como.

Archivo|Guardar Como Guarda los datos actuales en un archivo nuevo. Este archivo se tomará como el archivo actual.

Archivo|Limpiar Inicializa a cero todos las casillas de programación.

Archivo|Salir Cierra el programa.
